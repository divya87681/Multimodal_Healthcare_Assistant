# Multimodal AI-Driven Healthcare Assistant

This project is an AI-based ICU monitoring system that uses computer vision, audio processing, and NLP.

## Features
- Facial pain detection using Action Units
- Agitation detection using pose estimation
- Emergency speech detection using Whisper ASR
- Multimodal risk scoring system
- RAG-based medical assistant with safety validation

## Technologies
Python, OpenCV, MediaPipe, Whisper, Transformers
